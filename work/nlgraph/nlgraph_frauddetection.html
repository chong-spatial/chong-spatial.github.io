This is fraud detection graph with Node-Link Graph JavaScript library - NLGraph.js

1. Find accounts linked to 57#cc (the central node) which is a suspect account styled in a differen shape and fill color

2. Filter account type, only display mobible, credit card, and email. (Node filter)

3. Filter associations of transcation, association can not be of type A, B, or C (Edge filter)

4. Filter the relatedness score, can not be smaller than 0.5

Node label is account ID + '#' + account type; link label is transaction type. This graph allows users to immediately see suspect account and associated trasactions, fraud propagation, and how the fraudulent transactions are identified.
